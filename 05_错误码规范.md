# 错误码与报错规范

版本：v1.0　　日期：2026-05-01

## 1. 错误码编码规则

错误码格式：`[层级前缀]-[模块代码]-[错误序号]`

示例：`HW-SOIL-001` = 硬件层(HW) 土壤传感器(SOIL) 第 1 号错误

| 层级前缀 | 含义 |
|---------|------|
| `HW` | 硬件层（传感器/执行器/供电） |
| `NET` | 网络层（WiFi/HTTP 通信） |
| `MDL` | 模型层（视觉/音频推理） |
| `AGT` | Agent 层（推理/决策） |
| `ACT` | 执行层（继电器/确认机制） |
| `SYS` | 系统层（主机/数据库/日志） |

## 2. 严重等级定义

| 等级 | 标识 | 含义 | 响应动作 |
|------|------|------|---------|
| CRITICAL | P0 | 系统无法继续运行，可能造成作物损失 | 立即停止所有执行器，推送紧急告警 |
| ERROR | P1 | 某模块功能失效，影响感知或执行 | 降级运行，推送告警，记录日志 |
| WARN | P2 | 数据异常或性能下降，不影响核心功能 | 记录日志，推送提示 |
| INFO | P3 | 正常运行信息，用于监控和调试 | 仅记录日志 |

## 3. HW 硬件层错误码

| 错误码 | 等级 | 描述 | 触发条件 | 处理方式 |
|--------|------|------|---------|---------|
| HW-SOIL-001 | P1 | 土壤湿度读数异常 | 读数 < 0 或 > 100 | 丢弃本次读数，使用上次值，连续 3 次触发推送告警 |
| HW-SOIL-002 | P2 | 土壤湿度传感器漂移 | 同环境下读数波动 > ±15% | 标记数据可信度低，记录日志 |
| HW-TEMP-001 | P1 | DHT22 读取超时 | OneWire 无响应 > 2s | 重试 3 次，失败后推送 WARN，跳过本轮 |
| HW-LIGHT-001 | P2 | 光照值超量程 | 读数 > 65535 lux | 截断为最大值，记录日志 |
| HW-MIC-001 | P1 | 麦克风无信号 | I2S 数据全零 > 5s | 重启 I2S 外设，推送告警 |
| HW-CAM-001 | P1 | 摄像头初始化失败 | 上电后无 JPEG 输出 | 重启摄像头，失败 3 次后进入无视觉降级模式 |
| HW-CAM-002 | P2 | 图像模糊/过曝 | 亮度均值 < 20 或 > 230 | 调整曝光参数，重拍一次 |
| HW-PWR-001 | P0 | 供电电压过低 | 电池电压 < 3.2V | 立即关闭摄像头和执行器，仅保留传感器采集和告警 |
| HW-PWR-002 | P2 | 供电电压警告 | 电池电压 < 3.5V | 推送低电量提示，降低采集频率 |

## 4. NET 网络层错误码

| 错误码 | 等级 | 描述 | 触发条件 | 处理方式 |
|--------|------|------|---------|---------|
| NET-WIFI-001 | P1 | WiFi 连接断开 | 连续 5 次心跳无响应 | 启动指数退避重连，本地缓存数据待上报 |
| NET-WIFI-002 | P2 | WiFi 信号弱 | RSSI < -80 dBm | 记录日志，适当降低上报频率 |
| NET-HTTP-001 | P1 | 主机 HTTP 接口无响应 | POST 超时 > 10s | 本地缓存，重试队列，推送告警 |
| NET-HTTP-002 | P2 | HTTP 返回非 200 状态 | 4xx / 5xx 响应 | 记录响应码和请求体，不重试 4xx |
| NET-SYNC-001 | P2 | 时间同步失败 | NTP 同步超时 | 使用本地 RTC 时间，标记时间戳为"未校准" |

## 5. MDL 模型层错误码

| 错误码 | 等级 | 描述 | 触发条件 | 处理方式 |
|--------|------|------|---------|---------|
| MDL-VIS-001 | P1 | 视觉模型推理超时 | 推理耗时 > 30s | 丢弃本帧，记录超时，检查 Ollama 状态 |
| MDL-VIS-002 | P2 | 视觉模型输出为空 | 返回空字符串或 null | 使用默认描述"视觉感知暂不可用"，继续传感器推理 |
| MDL-AUD-001 | P1 | 音频模型加载失败 | 模型文件缺失或损坏 | 降级为无音频分析模式，推送告警 |
| MDL-AUD-002 | P2 | 音频分类置信度过低 | 最高类别置信度 < 0.5 | 输出"环境音不确定"，不用于 Agent 决策 |
| MDL-AUD-003 | P2 | 音频输入过短 | 音频帧 < 0.5 秒 | 等待下一个完整窗口，丢弃当前帧 |

## 6. AGT Agent 层错误码

| 错误码 | 等级 | 描述 | 触发条件 | 处理方式 |
|--------|------|------|---------|---------|
| AGT-LLM-001 | P1 | LLM 推理超时 | 推理耗时 > 60s | 中止本轮，推送告警，记录上下文 |
| AGT-LLM-002 | P1 | LLM 输出 JSON 解析失败 | 返回非法 JSON 格式 | 重试一次，失败则跳过本轮执行 |
| AGT-LLM-003 | P2 | 输出动作不在白名单 | action 字段为未知值 | 拒绝执行，记录异常输出内容 |
| AGT-CTX-001 | P2 | 上下文组装缺失字段 | 某路传感器数据缺失 | 用默认值填充并标注"数据缺失"，继续推理 |

## 7. ACT 执行层错误码

| 错误码 | 等级 | 描述 | 触发条件 | 处理方式 |
|--------|------|------|---------|---------|
| ACT-REL-001 | P0 | 继电器执行超时未断开 | GPIO 高电平持续超过最大时长 | 强制拉低 GPIO，记录 P0 告警，推送紧急通知 |
| ACT-REL-002 | P1 | 继电器状态反馈异常 | 电流检测值与预期不符 | 停止执行，推送检查提示 |
| ACT-CFM-001 | P1 | 高风险操作确认超时 | 手机未在 60 分钟内确认 | 自动取消，记录日志，重新推送提示 |
| ACT-CFM-002 | P2 | 确认消息发送失败 | Webhook 推送失败 | 重试 3 次，失败后禁止该操作执行 |
| ACT-LIM-001 | P1 | 单日操作次数超限 | 同类操作 > 阈值次/天 | 锁定当日该操作，推送异常告警 |

## 8. SYS 系统层错误码

| 错误码 | 等级 | 描述 | 触发条件 | 处理方式 |
|--------|------|------|---------|---------|
| SYS-DB-001 | P1 | SQLite 写入失败 | 磁盘空间不足或文件锁 | 推送告警，清理 30 天前旧日志 |
| SYS-MEM-001 | P2 | 主机内存不足 | 可用内存 < 512MB | 推送警告，停止视觉模型直到内存释放 |
| SYS-GPU-001 | P1 | GPU 显存溢出 | CUDA out of memory | 重启 Ollama 服务，降级至 CPU 推理 |
| SYS-LOG-001 | P3 | 日志文件过大 | 单文件 > 100MB | 自动轮转，压缩归档 |

## 9. 错误日志标准格式

所有错误以 JSON Lines 格式写入 `/logs/system.log`：

```json
{
  "timestamp": "2026-05-01T08:23:11Z",
  "code": "HW-SOIL-001",
  "level": "P1",
  "node_id": "field_01",
  "value": -1,
  "message": "土壤湿度读数超出范围",
  "action": "skip_and_alert"
}
```

## 10. 降级运行策略

| 故障模块 | 降级策略 | 恢复条件 |
|---------|---------|---------|
| 视觉模型不可用 | 仅使用传感器 + 音频数据进行 Agent 推理 | 视觉模型服务恢复响应 |
| 音频模型不可用 | 仅使用视觉 + 传感器数据 | 音频模型文件修复或重载 |
| WiFi 断线 | 本地缓存数据，恢复后批量上报 | WiFi 重连成功 |
| 主机宕机 | 继电器保持当前状态（默认常闭阀关闭） | 主机服务重启 |
| 电量过低 | 关闭摄像头/执行器，仅保留传感器和网络 | 电池电压 > 3.6V |
